ECE 2713 Design Project

Digital Signal Processing & Audio Filtering

Project Overview

Professional digital audio processing using MATLAB for signal analysis, filter design, and noise reduction. Demonstrates advanced DSP concepts including Butterworth filters, frequency domain analysis, and real-time audio processing.

Pure Tone Generation & Analysis

Creating and analyzing digital audio signals with specific frequencies

MATLAB Implementation

% Problem 1a: 440 Hz Pure Tone
Fs = 44100;           % Sampling frequency
N = Fs * 2;          % 2-second signal
n = 0:N-1;           % Time vector
f_analog = 440;      % Analog frequency
w_dig = 2*pi*f_analog/Fs;  % Digital frequency
x = cos(w_dig * n);  % Pure tone signal

% Normalize and play
x = x / max(abs(x));
sound(x, Fs, 16);

% Frequency domain analysis
X = fftshift(fft(x));
Xmag = abs(X);
XmagdB = 20*log10(Xmag);

% Plot frequency response
w = -pi:2*pi/N:pi-2*pi/N;
f = w * Fs / (2*pi);
figure(1);
plot(f, XmagdB);
title('Centered DFT Magnitude for 440 Hz Pure Tone');
xlabel('Analog Frequency (Hz)');
ylabel('Magnitude (dB)');

Key Results

  • Generated 2-second 440 Hz pure tone (A4 note)
  • Sampling rate: 44.1 kHz (CD quality)
  • Frequency domain analysis using FFT
  • Successfully demonstrated audio playback and file I/O

Digital Butterworth Filter Design

Lowpass and highpass filter implementation for signal separation

Interactive Filter Demonstration

Passband Edge: 250 Hz
Stopband Edge: 1000 Hz
Filter Order: 12

Filter Design Code

% Butterworth Filter Design
Wp = 0.4;    % Normalized passband edge
Ws = 0.6;    % Normalized stopband edge
Rp = 1;      % Passband ripple (dB)
Rs = 60;     % Stopband attenuation (dB)

[Nf, Wn] = buttord(Wp, Ws, Rp, Rs);
[num, den] = butter(Nf, Wn);

% Apply filter
y = filter(num, den, x);

% Frequency response
freqz(num, den, 1024);

Noise Reduction Implementation

% Read noisy signal and noise sample
[noisy_sig, Fs] = audioread('noisysig.wav');
[noise_samp, ~] = audioread('noisesamp.wav');

% Analyze frequency spectra
X_noisy = fftshift(fft(noisy_sig));
X_noise = fftshift(fft(noise_samp));

% Design lowpass filter based on analysis
Wp = 0.3;  % Passband edge (normalized)
Ws = 0.5;  % Stopband edge (normalized)
Rp = 1;    % Passband ripple
Rs = 60;   % Stopband attenuation

[Nf, Wn] = buttord(Wp, Ws, Rp, Rs);
[num, den] = butter(Nf, Wn);

% Apply filter
filtered_sig = filter(num, den, noisy_sig);
filtered_sig = filtered_sig / max(abs(filtered_sig));

% Save filtered signal
audiowrite('filteredsig.wav', filtered_sig, Fs);
